# Pong (Godot)

Um clone simples do clássico **Pong**, desenvolvido em **Godot Engine** como exercício de aprendizado de desenvolvimento de jogos.

O projeto implementa os principais elementos do jogo original:

- Controle do jogador
- IA do oponente
- Física de colisão da bola
- Sistema de pontuação
- Efeitos sonoros
- Screen shake
- Export para Web (HTML5)

🎮 **Jogue no navegador:**  
https://fabiolrodriguez.itch.io/pong


# Tecnologias usadas

- **Godot Engine**
- **GDScript**
- Export Web (HTML5 / WebAssembly)
